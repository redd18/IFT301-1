To determine the data rate for 16 QAM using a \(\frac23\) coding rate with 2 parallel data stream

a) Assume that there are 64 subcarriers to find the symbol rate on each sub-channel. Therefore

subcarrier symbol\(= 16 QAM\)
coding rate \(= \frac23\)
\(\)
\(\) $$T_N = \frac{64}{20*10^6}$$ $$T_N = 3.2 µs$$ The number of coded bits for subcarrier symbol for 16 QAM is $$log_216 = log_22^4 = 4$$ Note that there are a total of 48 subcarriers used for data transmission.

Therefore, the data rate is $$R = 4*48*\frac23*\frac{1}{3.2*10^{-6}}$$ $$R = 40 Mbps$$

b) Data rate for 50 MHz channel $$T_N = \frac{64}{40*10^6}$$ $$T_N = 1.6 µs$$ $$R = 4*48*\frac23*\frac{1}{1.6*10^{-6}}$$ $$R = 80 Mbps$$
